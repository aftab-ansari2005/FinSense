🎨 Model Metrics Display

Performance Indicators:

  • Version: Model version tag (e.g., v1.0)
  • Accuracy: Color-coded percentage
    • ≥90% = Excellent (green)
    • ≥75% = Good (blue)
    • ≥60% = Fair (yellow)
    • <60% = Poor (red)
  • MAE: Mean Absolute Error in dollars
  • RMSE: Root Mean Square Error in dollars
  • Last Updated: Timestamp of last model update
